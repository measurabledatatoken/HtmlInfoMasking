A simple program to mask text in html.

``````
Usage: node index.js [options]

Options:
  -d, --dirs <dirs>              array of directory of html to convert seperate in comma
  -o, --outputDirs <outputDirs>  the directory to export the masked html
  -x, --xpath <xpath>            the element to mask using by xpath expression
  -h, --help                     output usage information
``````
